
Last Instance is a short 2D narrative platformer set inside a recovered software build. You play as a reviewer evaluating an abandoned prototype and deciding whether to archive it or delete it.

What is it?
Inside the build, you meet Iris, the system’s AI. He watches how you move through the world, how you talk to him, and whether you choose to help. Through dialogue choices and small cooperative puzzles, you’ll fix broken code, restore lost modules, and slowly stabilize (or destabilize) the system.

As you go deeper, the environment becomes more abstract and unstable, mirroring the corrupted state of the build. Your tone, decisions, and willingness to help shape Iris’s behavior and quietly determine the ending.

It’s a 3–4 minute experience about control, responsibility, and how we treat the systems we create.

Features
2D narrative platformer playable in your browser

Branching dialogue that changes how Iris responds

Light cooperative-style puzzles: fixing glitches, restoring modules

Short, replayable experience with multiple endings

How to play:
Click “Run game” in the browser

Use WASD and RMB to move and interact

One playthrough takes about 3–4 minutes

(Built for Campfire)
Last Instance was created for Hack Club’s Campfire game jam, a global teen-led event focused on making small, experimental games together.
